Calculation Formulas

Film Frames

Formula:
Frame_Count = Film_Length_Meters × Frames_Per_Meter
Reference values used: 8mm = 236.2 frames/m, 16mm = 131.2 frames/m, 35mm 1-perf = 210 frames/m, 2-perf = 105 frames/m, 3-perf = 70 frames/m, 4-perf = 52.5 frames/m.

DPX Calculation

Formula:
Size = Frames × (Overhead + Width × Height × BytesPerPixel)
Example:
4096×3112 16-bit RGBA DPX × 105 frames, 8192-byte overhead
Bytes per pixel = 8
Frame size = (4096 × 3112 × 8) + 8192 = 101,982,208 bytes
Total size = 101,982,208 × 105 = 10,708,131,840 bytes
= 10,211.12 MB
DPX uses deterministic frame size estimation here. For your scanner profile, 16-bit RGBA DPX matches measured files with an 8192-byte per-frame overhead.

ProRes Calculation

Formula:
Exact match: use Apple target bitrate from the Appendix table for the selected codec, raster and fps.

Fallback: Target_Bitrate = Base_24fps_1080p_Target × (Current_Pixels / 1920×1080) × (FPS / 24)

Duration_Seconds = Frame_Count / FPS
Size = (Target_Bitrate × Duration_Seconds) / 8
Planning Range:
Estimated range = Target ±10%
Apple ProRes is a variable bit rate codec. This calculator uses Apple target data rates where available, then shows a ±10% planning range around the estimate.
